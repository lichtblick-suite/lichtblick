# SigninForm

未ログインユーザー向けのサインインフォームコンポーネント

## 概要

`SigninForm`は、未ログインユーザーに対してFoxgloveアカウントの価値提案を表示し、サインインを促すコンポーネントです。視覚的なグラフィックと明確な機能説明により、ユーザーのアカウント作成を促進します。

## 主な機能

### ✅ 価値提案の表示
- レイアウト同期機能の説明
- レイアウト共有機能の説明
- ロボティクスデータ管理機能の説明

### ✅ 視覚的な訴求
- `AccountSyncGraphic`による同期概念の視覚化
- 中央配置による目立つ表示

### ✅ アクション促進
- プライマリカラーのサインインボタン
- 明確なCTA（Call to Action）

## Props

このコンポーネントはPropsを受け取りません。

## 使用例

```tsx
import SigninForm from './SigninForm';

function AccountSettings() {
  const { currentUser } = useCurrentUser();

  if (!currentUser) {
    return <SigninForm />;
  }

  return <AccountInfo currentUser={currentUser} />;
}
```
