# AccountSyncGraphic

アカウント同期を視覚的に表現するSVGグラフィックコンポーネント

## 概要

`AccountSyncGraphic`は、デバイス間でのデータ同期を象徴するSVGグラフィックコンポーネントです。地球儀風の円形パターンと2つのデバイスを組み合わせて、アカウント同期の概念を視覚的に表現します。

## 主な機能

### ✅ 同期概念の視覚化
- 地球儀風の円形グリッドパターン
- デバイス間の接続性表現
- 装飾的な点群による接続感の演出

### ✅ デバイス表現
- 左下：ノートパソコン風デバイス
- 右上：デスクトップ風デバイス
- 各デバイスの接続範囲表示

### ✅ テーマ対応
- `currentColor`による色彩制御
- Material-UIテーマとの自動連動
- 一貫したビジュアルデザイン

## デザイン原則

### 視覚的メタファー
- **地球儀**: グローバルな接続性
- **デバイス**: 具体的な使用端末
- **点群**: データの流れ
- **円形**: 完全性と統合

### 色彩設計
- **currentColor**: テーマとの自動連動
- **単色表現**: シンプルで洗練された外観
- **線画スタイル**: 軽量で明瞭な表現

## 技術仕様

### SVG viewBox
```svg
<svg width={width} viewBox="0 0 256 256">
```
- 256x256の正方形キャンバス
- スケーラブルなベクターグラフィック
- レスポンシブ対応

### ストローク設定
- **strokeWidth**: 4.5px（統一された線幅）
- **strokeLinecap**: "round"（丸い線端）
- **fill**: "none"（線画表現）

## パフォーマンス

### 軽量な実装
- **静的SVG**: 動的生成不要
